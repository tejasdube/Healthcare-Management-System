# Healthcare Management System

Welcome to the Healthcare Management System repository! This project aims to revolutionize healthcare delivery by leveraging technology to streamline operations and enhance patient care.

## Overview

The Healthcare Management System is a dynamic web application built using JSP, Servlets, and JDBC. It offers a comprehensive solution for managing patient records, appointments, and medical data in healthcare institutions.

## Features

- **User Authentication:** Secure login and authentication system for admins, doctors, and patients.
- **Patient Management:** Manage patient records, including personal information, medical history, and appointments.
- **Appointment Booking:** Allow patients to schedule appointments with healthcare providers.
- **Doctor Dashboard:** Provide doctors with access to patient information and appointment schedules.
- **Admin Panel:** Administrative interface for managing users, appointments, and system settings.

## Technologies Used

- JavaServer Pages (JSP)
- Servlets
- Java Database Connectivity (JDBC)
- HTML/CSS/Bootstrap for frontend
- MySQL for database management

## Getting Started

1. Clone the repository: `git clone https://github.com/tejasdube/Healthcare-Management-System`
2. Install Eclipse IDE (https://www.eclipse.org/downloads/)
3. Install Mysql Workbench (https://dev.mysql.com/downloads/workbench/) setup -> file -> open sql script -> select database file provided in code -> run script
4. Install apache tomcat 9 (https://tomcat.apache.org/download-90.cgi) configure it -> Open Eclipse -> window -> show view -> server ->select server
5. Open Eclipse >>> File >> Open Folder >> Select Project Folder >> Run on Server
